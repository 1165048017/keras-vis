ImageModifier

Abstract class for defining an image modifier. An image modifier can be used with the Optimizer.minimize to make pre and post image changes with the gradient descent update step.

modifier.pre(img)
# gradient descent update to img
modifier.post(img)

ImageModifier.post

post(self, img)

Implement post gradient descent update modification to the image. If post-processing is not desired, simply ignore the implementation. It returns the unmodified img by default.

Args:

  • img: An N-dim numpy array of shape: (samples, channels, image_dims...) if data_format='channels_first' or (samples, image_dims..., channels) if data_format='channels_last'.

Returns:

The modified post image.

Jitter

Jitter.__init__

__init__(self, jitter=16)

Implements an image modifier that introduces random jitter in pre and undoes in post. Jitter has been shown to produce crisper activation maximization images.

Args:

  • jitter: Number of pixels to jitter in rows and cols dimensions.
